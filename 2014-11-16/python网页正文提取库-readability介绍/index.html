python网页正文提取库-readability介绍

在爬虫过程中我们经常需要获得网页的正文，对于同一网站的页面我们可以直接通过分析页面结构来获取正文内容。但是如果不同网站数量太多，这时候再一个个分析网页结构无疑是非常麻烦的，前几天在抓正文时遇到了这个问题，搜索后找到了一个比较好用的python库: python-readability

readability利用lxml库进行网页分析然后获取到网页内容，所以需要先安装好lxml,然后直接运行安装:

pip install readability-lxml
(may need install cssselect)

更详细的用法可以直接参考源代码readability.py
